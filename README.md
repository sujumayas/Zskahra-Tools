# Zskahra-Tools

Herramientas web para el mundo de **Zskahra** — suplemento homebrew para HARP.

## Herramientas

- **Explorador del mundo** — Villas, sistema social, sufijos de hito y estructura del mundo.
- **Generador de nombres** — Nombres para villagers con raza, villa, sufijo, edad y trasfondo.
- **Oráculo de Clima** — Genera el clima por khar para cualquier bioma.
- **Guía del Herbalista** — Sistema de recolección de hierbas, ingredientes y preparaciones.

## Despliegue

El sitio se publica en [Netlify](https://netlify.com) como archivos estáticos. No requiere build — se sirve directamente desde la raíz del repositorio.
